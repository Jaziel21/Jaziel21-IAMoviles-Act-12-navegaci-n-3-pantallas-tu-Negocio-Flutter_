# Blueprint: Librería AJMG

## Overview

Librería AJMG is a simple Flutter application that allows users to browse a list of books, view offers, and simulate a book purchase.

## Features

*   **Book Listing:** The main screen displays a list of books with their titles, prices, and images.
*   **Offer Page:** A dedicated page showcases special offers on books.
*   **Shopping Cart:** A simple shopping cart page confirms the purchase of a book.
*   **Navigation:** The application uses named routes for navigation between the different pages.

## Design

*   **Theme:** The app uses a custom dark theme with a yellow accent color.
*   **Fonts:** The app uses the `google_fonts` package for custom fonts.
*   **Layout:** The layout is built using basic Flutter widgets like `Scaffold`, `AppBar`, `ListView`, `Column`, and `Row`.
*   **Images:** The app uses local images from the `imagenes` directory.
